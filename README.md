# Restaurant-web-App

This Resturant Odering Application is a platform where client/customers can order avaliable meals on the menu at their convenience and have it served to them from the cofort of their mobile devices.

The features in it include:
A minimal responsive design
A simple UI interface for users

The platform has three pages which include:
index.html
cart.html
checkout.html

The tech stack used weas HTML, Bootstrap, CSS and Vanilla JavaScript 

HOW THE APP IS USED

Once the user enters the home page, upon scrolling the menu is seen, where they can place their order. 
When dine, the go to the cart page to view their choices.
And finally move to checkout for their invoice.
